# 🐶 AdotaAí

Seu melhor amigo pode estar aqui.

O AdotaAí é uma plataforma de adoção de pets criada para conectar pessoas que querem doar animais com pessoas que desejam adotar.

## Objetivo

Facilitar a adoção responsável de cães e gatos.

## Como funciona

- Usuários podem cadastrar pets para adoção
- Outros usuários podem visualizar os pets
- É possível entrar em contato com o responsável

## Cidade inicial

Campos dos Goytacazes - RJ
